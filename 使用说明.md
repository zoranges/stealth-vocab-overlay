# 工位背单词悬浮窗

一个在桌面角落低调显示单词的悬浮窗，适合上班/上课时偷偷背单词。
基于 Windows PowerShell + WPF，**无需安装任何环境**，Win10 及以上双击即用。

## 快速开始

1. 把整个文件夹复制到电脑任意位置（不要只复制单个文件）。
2. 双击 `启动背单词悬浮窗.bat`。
3. 桌面出现单词悬浮窗，默认自动轮播。

> 首次运行会自动生成设置文件和熟悉/生词库文件，属于正常现象。

## 常用操作

- **点击悬浮窗且不拖动**：打开设置。
- **右键悬浮窗**：打开设置。
- **按住左键拖动悬浮窗任意位置**：改变位置。
- **双击悬浮窗 / 空格 / 方向右**：下一个单词。
- **方向左**：上一个单词。
- **滚轮**：在专注模式下切换单词。
- `F8`：全局热键下一词（可在设置里改成 F6–F12、PageDown 或 Insert）。
- `F6`：默认一键隐匿/显示悬浮窗（可在设置里改）。

## 设置说明

设置窗口里的选项实时生效，自动保存到 `stealth_vocab_wpf_settings.json`。

- **文字预设**：一键切换配色方案。
  - 白字清晰 / 黑字清晰：高对比，适合大多数壁纸。
  - 灰字隐蔽：低调不显眼。
  - 高对比描边：文字带光晕，复杂壁纸上最清晰。
- **清晰增强**：开关文字描边光晕（根据字色自动选反色阴影，任意背景都清晰）。
- **透明度**：最低可调到接近不可见，适合需要极低存在感的场景。
- **一键隐匿按键**：默认 `F6`，按一次隐藏悬浮窗，再按一次显示。
- **本次背诵词库**：全部 / 未分类 / 熟悉 / 生词。
- **开始序号**：输入当前词库里的序号并点击“跳到这里”，可以指定本次从哪里开始。
- **自动续背**：程序会记住每个词库上次背到的位置，切换词库或下次启动时继续。
- **导入词库 JSON**：点击“导入熟悉词库 JSON”或“导入生词库 JSON”，可选择外部 JSON 文件导入，程序会自动刷新未分类/熟悉/生词目录。
- **专注模式**：滚轮切词，左键标为熟悉，右键标为生词；也可自定义熟悉/生词快捷键，默认 `A` 熟悉、`D` 生词；按 `Enter` 打开设置，按 `Esc` 退出。
- **词表目录**：分成“全部 / 未分类 / 熟悉词库 / 生词库”四个独立标签页，每个标签页单独显示自己的单词；熟悉词为绿色，生词为红色加粗。

## 数据文件

| 文件 | 说明 | 是否含个人数据 |
|------|------|----------------|
| `stealth_vocab_wpf.ps1` | 主程序 | 否 |
| `启动背单词悬浮窗.bat` | 启动器 | 否 |

| `生词库.json` | 你标记为生词的词 | **是**（自动生成） |

## 分享给他人

双击 `打包发布.ps1`，会自动生成一个 zip 文件（如 `工位背单词悬浮窗_20260620.zip`），
**只包含运行所需的 5 个文件，不含你的设置和熟悉/生词记录**。
把这个 zip 发给别人，对方解压后双击 `启动背单词悬浮窗.bat` 即可使用。

## 常见问题

**Q：窗口一闪而过 / 没反应？**
确认整个文件夹完整复制（含 `stealth_vocab_wpf.ps1` 和 `红宝书词汇.json`）；
检查杀毒软件是否拦截了 PowerShell 脚本。

**Q：文字发虚/看不清？**
打开设置，点"高对比描边"预设，或勾选"清晰增强"开关。

**Q：换电脑后熟悉/生词记录没了？**
熟悉/生词记录保存在 `熟悉词库.json` 和 `生词库.json`，换电脑时把这两个文件一起复制过去即可。

**Q：系统要求？**
Windows 10 及以上（自带 PowerShell 5.1 与 WPF）。
